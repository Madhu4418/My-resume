Hangman project in c++

c++

It consists of a stick man. The initial state consists of a random word with dashes to fill. It takes the user's input key and checks if that letter is present in the word. If it is there, The letter gets entered in the word, and if it doesnt, then the stickman grows a leg. Every wrong choice will lead to the growth of a part of the stickman's body. When the stickman's body is complete, then this is the final state in which the game is over. This denotes that the user has run out of chances to predict the word and the computer wins.
If the user guesses the correct word, another similar word is displayed on the screen for the user to guess.
Project link: https://github.com/Madhu4418/HangmanC-

Medicos - Medical dictionary app


Android Application

Its not just a dictionary app. Its got added features:
•When a name of a disease is clicked, it shows the description, symptoms, causes, diagnosis, treatment of a particular disease.
•It has a feedback module. The user can enter the feedback about the app and it will be sent to the developer’s mail id.(I have hardcoded my email id in the java email service)
•It has a small module that describes what all this app is used for and its features.
•It takes in the location of the user and finds out pharmacies near them using google maps API.
•When a data is not available on a particular disease, there is a module where the user can enter the disease to be searched for and the details regarding that disease will be fetched.
•Users can enter their symptoms and the app will give back the names of the disease that they could possibly have.
